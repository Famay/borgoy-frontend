# VerMeat

VerMeat - сервис для проверки происхождения партий боргойской баранины и подлинности сопроводительных сертификатов.

## Что можно проверить

- статус сертификата;
- номер партии;
- номер сертификата;
- данные о поставщике;
- дату регистрации;
- цифровой отпечаток сертификата.

## Как работает проверка

Проверка доступна по QR-коду, номеру партии или номеру сертификата. Если данные найдены, сервис показывает информацию о партии и статусе сертификата.

## Назначение

Сервис помогает покупателям, партнерам и контролирующим сторонам быстро убедиться, что партия продукции зарегистрирована и связана с проверяемым сертификатом.
